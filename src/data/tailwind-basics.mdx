## âš™ï¸ Installation (Quick Setup)

Getting started is easy:

```bash
npm install -D tailwindcss
npx tailwindcss init
```

In your CSS file (e.g., `index.css`), add the following:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Then update your `tailwind.config.js`:

```js
module.exports = {
  content: ["./src/**/*.{js,ts,jsx,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

Finally, import your CSS in the root of your app (e.g., `_app.tsx` if you're using Next.js).
